原型链继承

子类的原型指向父类的对象

缺点: 引用类型的成员会被所有子类的对象共享

借用构造函数

利用apply函数拷贝父类的成员

缺点：无法继承父类原型上的成员

组合继承

结合原型链继承和借用构造函数

缺点：父类的构造函数被调用了两次

寄生组合继承

  1. 借用构造函数
  2. 创建一个辅助函数使其原型指向父类的原型，新建一个该辅助函数的对象(该对象继承父类原型)
  3. 将该对象的constructor属性指向子类
  4. 将该对象作为子类的原型
